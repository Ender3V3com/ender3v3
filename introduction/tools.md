To make the tinkering process on the printer easier and more pleasant, we recommend you to get these tools:

- Allen Wrenches set
    - torx screwdriver set
- Caliper
- **Multimeter**
- 6mm socket for nozzle change
- Pliers & Cutters
- M2x12 socket head screws
- Deburring tool
- Bed Scraper
- Scalpel/Xacto knife
